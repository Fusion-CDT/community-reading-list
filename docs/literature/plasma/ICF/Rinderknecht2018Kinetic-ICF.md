This paper is a great review on kinetic effects in ICF, with a significant number of references which are also very useful. It outlines which kinetic effects are relevant to each stage in an ICF implosion, references some important experimental results, and then goes into detail outlining theory and simulation techniques. It is a great reference for anyone doing anything related to kinetic modelling in ICF.
